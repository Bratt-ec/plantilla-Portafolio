WEB DE IST "JOSÉ OCHOA LEÓN"

En el 2018 tuve la oportunidad de participar en el diseño de la interfaz de la web de Instituto Superior Tecnológico Superior "José Ochoa León", la misma que sería mostrada al público en la Feria Tecnológica del año 2019. Está desarrollado en Bootstrap y JQuery con diseño adaptable a difrentes dispositivos.

SISTEMA WEB DE CLÍNICA

Este es un sistema web con responsive design desarrollado con PHP con programación orientada a objetos, HTML, CSS, Bootstrap, JQuery y base de datos MySQL. Por el momento permite la administración de citas médicas y listado de pacientes. Ya le iré agregando fucionalidades.

WEB DE CONTROL DE LIBROS

Es una página web informativa con responsive design y que permite hacer la reservación previa de libros de la biblioteca de la Asociación de Obreros “Unión y Disciplina”. Fue desarrollado usando lenguaje de programación PHP, librería JQuery, Html, Bootstrap y una base de datos en PostgreSQL.

SISTEMA DE ADMINISTRACIÓN

Estos sitios web que forman parte del módulo de matriculas del Sistema del ISTJOL esta desarrollado en PHP, JavaScript, la libreria JQuery, AdminLTE y una base de datos PostgreSQL.
